Bayesian Multi-Arm Treatment Selection Extension
Overview

This repository contains the code used to replicate and extend the Bayesian treatment selection framework proposed by Komaki et al. for Phase II clinical trials.

The original methodology was developed for two-arm treatment comparisons and relies on a threshold-based decision rule constructed from posterior probabilities of clinically meaningful treatment differences. This project reproduces the original simulation study and proposes a multi-arm extension that allows treatment selection among an arbitrary number of competing treatment arms.
